# Stellar Keystone Examples

Example projects demonstrating RBAC integration patterns.

## Available Examples

### [secure_vault/](./secure_vault/)

Demonstrates composing RBAC with a vault contract for withdrawal authorization.

**Key Features:**
- RBAC contract integration via cross-contract calls
- Role-based withdrawal limits
- On-chain authorization enforcement

**Use Case:** Treasury systems, payout automation, multi-signature patterns

See [secure_vault/README.md](./secure_vault/README.md) for details.

## Running Examples

Each example has its own README with specific build and deployment instructions.

## Contributing Examples

Examples should demonstrate **real-world integration patterns**, not just API usage.

Good examples show:
- How to compose RBAC with other contracts
- Common authorization patterns
- Production-ready error handling
- Security best practices
